```

─────────────────────────────────────────────────────

```

### 시간당 비용 계산

```
시간당 비용 = $0.00001544/초 × 3,600초 = $0.055584/시간
```

### 일당 비용 계산

```
일당 비용 = $0.055584/시간 × 24시간 = $1.334/일

## $5 크레딧으로 사용 가능한 시간

### 1. 연속 실행 기준

```
$5 ÷ $0.055584/시간 = 약 90시간
```

**결론: 약 90시간 (약 3.75일) 연속 실행 가능**

### 2. 월간 사용 가능 시간 (Hobby 플랜 기준)

Hobby 플랜은 **$5/월 구독료 + $5 크레딧**을 제공합니다.

```
월간 크레딧: $5
시간당 비용: $0.055584
─────────────────────────────
월간 사용 가능 시간: 약 90시간
```

**하지만 실제로는:**
- 앱이 항상 켜져 있지 않을 수 있음
- 사용하지 않을 때는 비용이 발생하지 않음
- 실제 사용 시간만 과금됨

### 3. 실제 사용 시나리오별 계산

#### 시나리오 1: 하루 8시간만 사용 (업무 시간)
```
일일 비용 = $0.055584/시간 × 8시간 = $0.44/일
월간 비용 = $0.44/일 × 30일 = $13.2/월
$5 크레딧으로 = 약 11일 사용 가능
```

#### 시나리오 2: 하루 12시간 사용
```
일일 비용 = $0.055584/시간 × 12시간 = $0.67/일
월간 비용 = $0.67/일 × 30일 = $20/월
$5 크레딧으로 = 약 7.5일 사용 가능
```

#### 시나리오 3: 24시간 항상 켜져 있음
```
일일 비용 = $0.055584/시간 × 24시간 = $1.33/일
월간 비용 = $1.33/일 × 30일 = $40/월
$5 크레딧으로 = 약 3.75일 사용 가능

### Hobby 플랜 ($5/월) 포함 시

Hobby 플랜은 **$5/월 구독료**를 내고, **$5 크레딧**을 받습니다.

```
실제 월간 비용 = 구독료 $5 + (사용량 - $5 크레딧)
```

#### 예시 1: 24시간 항상 켜져 있음
```
월간 사용량: $40
크레딧: $5
─────────────────
실제 비용: $5 (구독료) + $35 (추가 사용량) = $40/월
```

#### 예시 2: 하루 8시간 사용
```
월간 사용량: $13
크레딧: $5
─────────────────
실제 비용: $5 (구독료) + $8 (추가 사용량) = $13/월
```

#### 예시 3: 하루 4시간 사용
```
월간 사용량: $6.6
크레딧: $5
─────────────────
실제 비용: $5 (구독료) + $1.6 (추가 사용량) = $6.6/월
```

## 참고사항

- **Egress (데이터 전송) 비용**: 
  - 일반적으로 적음 ($0.05/GB)
  - 파일 업로드/다운로드가 많으면 추가 비용 발생 가능

- **Storage (Volumes) 비용**:
  - 현재 사용 안 함
  - 필요 시 추가 비용 발생

- **자동 스케일링**:
  - Railway는 트래픽이 없을 때 자동으로 스케일 다운
  - 실제 사용 시간만 과금됨

- **무료 크레딧**:
  - 신규 사용자: $5 (30일 유효)
  - Hobby 플랜: 매월 $5 크레딧 제공
